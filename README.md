# 🍽 Recipe Blog Website

A responsive recipe blog website developed using HTML, CSS, and JavaScript.

## Features

* Display recipe cards with images
* Search recipes
* View full recipe details
* Responsive layout
* Modern card design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* Home Page
* Recipe Details Page
* Responsive Design

## How to Run

1. Download the project.
2. Open `index.html`.
3. Explore recipes.
